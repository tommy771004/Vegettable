# Assets

Place the following image files here:

- `icon.png` — App icon (1024x1024)
- `adaptive-icon.png` — Android adaptive icon (1024x1024)
- `splash-icon.png` — Splash screen icon (200x200 recommended)
- `favicon.png` — Web favicon (48x48)

Recommended: Use a green leaf or vegetable illustration with the app's green theme (#2E7D32).
